# xmlconfig
解析hadoop等相关项目的xml配置文件
xml配置文件格式如下
```
<?xml version="1.0" encoding="UTF-8"?>
<?xml-stylesheet type="text/xsl" href="configuration.xsl"?>
<configuration>
    <property>
        <name>name1</name>
        <value>value1</value>
        <tag>tag1,tag2</tag>
        <description>demo2</description>
    </property>
</configuration>
```
